# Strona do weryfikacji biznesowej (Google Play)

Plik strony znajduje się w `docs/index.html`.

## Publikacja przez GitHub Pages

1. Wypchnij zmiany do repozytorium na GitHub.
2. Wejdź w **Settings → Pages**.
3. W sekcji **Build and deployment** ustaw:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (lub Twoja główna gałąź)
   - **Folder**: `/docs`
4. Zapisz ustawienia i poczekaj na publikację.
5. Otrzymasz publiczny adres w formacie:
   `https://<twoj-login>.github.io/<nazwa-repo>/`

Ten adres możesz podać w procesie rejestracji/weryfikacji konta firmowego.
